Quick Install.

To install cealign, simply do the following: 
 > tar -jzvf cealign-VERSION.tar.bz2
 > cd cealign-VERSION
 > sudo python setup.py install
 > cat ./PYMOLRC >> ~/.pymolrc
The installer needs to edit their /.pymolrc file (or Mac/Windows equivalent) if s/he wants PyMol to automatically load the cealign  extension. Please edit the PYMOLRC file before executing the last command. Change, the directory from where PyMol should run the script.
